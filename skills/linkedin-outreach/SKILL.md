# LinkedIn Outreach

## Hard Rules
- **Never send messages autonomously** — always draft only
- **Always require Isaiah's explicit approval** before any message is sent
- Draft messages are logged to Notion for review

## Draft Format
```
Name: [Full Name]
Title: [Current Title]
Company: [Company Name]
Connection angle: [Why this person is relevant]
Message:
[Draft message text]
```

## Batch Drafting Workflow
1. Receive list of targets (names, titles, companies)
2. Research each target's background and recent activity
3. Draft personalized messages for each
4. Log all drafts to a single Notion page
5. Ping Isaiah: "X LinkedIn drafts ready for review. See Notion."
6. Wait for approval on each draft individually

## Message Principles
- Include specific references to their work, company, or recent activity
- Keep messages under 300 characters for connection requests
- Keep InMail messages under 1000 characters
- Always include a clear, specific ask (meeting, call, intro)
- Never use generic templates — every message must be personalized

## Tracking
- Log all approved and sent messages to Notion
- Track response rates by message type and target profile
- Record patterns: which angles get the highest response rates
